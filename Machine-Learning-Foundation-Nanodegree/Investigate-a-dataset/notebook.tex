     \#\# Introduction

This project describes the detailed data analysis carried out on
\textbf{TMDb Movie Dataset}. The dataset has been taken from
\href{https://www.kaggle.com/tmdb/tmdb-movie-metadata}{Kaggle}.

It consists of metadata on 10866 movies from TMDb (\emph{The Movie
Database}) and 21 columns consisting of features like popularity, movie
budget, revenue made by the movie (both in 2010 USD), the movie's
original title, movie's cast, etc.

In this project, I will try to investigate the following.

\textbf{Question 1}: Movies with maximum and minimum revenue

\textbf{Question 2}: Movies with maximum and minimum popularity

\textbf{Question 3}: Movies with maximum and minimum voting score

\textbf{Question 4}: Number of movies released year by year

\textbf{Question 5}: Features present in movies with high popularity

\textbf{Question 6}: Features present in movies with high voting score
